GUIAPI JavaDocs Version Index

Versions denote the version of GUIAPI the JavaDoc was built upon, not nescessarily the required version (although the versions should match)

'X' denotes a wildcard and '+' means it's explicitly compatible with minor versions greater than or equal to it (i.e. 1.5.0+ is compatible with 1.5.1 but NOT 1.6.0)
